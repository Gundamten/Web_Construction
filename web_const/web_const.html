剧情介绍

剑来

《剑来》是连载于纵横中文网中一部网络玄幻小说，作者是烽火戏诸侯。

该作品获得第四届橙瓜网络文学奖十大作品、百强作品、 最具潜力十大动漫IP。

2020年10月《剑来》获第二届泛华文网络文学“金键盘”奖—最佳故事创意奖。

大千世界，无奇不有。天道崩塌，我陈平安，唯有一剑，可搬山，断江，倒海，降妖，镇魔，敕神，摘星，摧城，开天。
