剧情介绍

《剑来》是烽火戏诸侯所著的长篇古典仙侠小说，主要讲述了骊珠洞天内小镇泥瓶巷少年陈平安自五岁成为孤儿后，艰难谋生，从在龙窑当学徒拉坯做瓷器，到被齐静春（文圣徒弟）选中代师收徒，他不断游历天下，提升自己的实力，逐步成长为文圣得意的关门弟子、剑气长城的末代隐官、落魄山的宗主的故事。该小说叙事宏大，不仅有神人妖魔鬼魅精怪，还有三教百家；不仅有儿女情长和侠肝义胆，也有世界源起、历史嬗变、宗教发展和族群对决，更有对文明的俯瞰、对人性的剖析和对人文的关怀。
